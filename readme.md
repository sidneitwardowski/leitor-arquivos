# Leitor Arquivos

### Requisitos
* Java 1.8+
* Maven 3.6.3+

#### Construir
```
mvn clean package
```

#### Executar Windows
```
java -jar target\leitorArquivos0.0.1-SNAPSHOT.jar
```

#### Executar Unix
```
java -jar target/leitorArquivos0.0.1-SNAPSHOT.jar
```
### Orientações

* Executar a aplicação e em seguida inserir o [Arquivo de modelo](modelo/modelo.dat) na pasta %HOMEPATH%/data/in.
